# mapify

Mapify is an open source project that will allow to users to convert any images to a Minecraft map.

## Features

- Make multi map art
- For Minecraft 1.13+
- More soon


## Dependecies

[JSzip](https://stuk.github.io/jszip/) by [Stuck](https://github.com/Stuk)
